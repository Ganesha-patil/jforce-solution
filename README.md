# 🗳️ Online Polling System

This is a **Spring Boot web application** for creating polls, voting, and viewing results.  
It includes user registration, login, poll creation, voting functionality, and result reporting.

---

## 🚀 Features

- User Registration and Login
- Create new polls
- Vote on available polls
- View poll results
- Report generation for polls
- Simple JSP-based UI
- MVC architecture using Spring Boot

---

## 🛠️ Technologies Used

- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- JSP (Java Server Pages)
- Maven
- MySQL (or any configured database)
